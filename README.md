# AutoHotkey 工具集

## 系统要求
- Windows 操作系统
- [AutoHotkey v2.x](https://www.autohotkey.com/) 或更高版本

## 安装步骤

### 1. 安装AutoHotkey
- 访问 https://www.autohotkey.com/
- 点击 **Download**
- 下载安装程序并运行
- 按照默认设置完成安装

### 2. 使用脚本
- 下载本仓库的 `.ahk` 文件
- 双击运行即可
- 或右键选择 "Run Script"

## 工具列表
- hide:双击桌面隐藏桌面图标
- close:双击enter（左右均可）实现息屏
